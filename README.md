# AnekdotVectors

Проверка как работает семантический поиск по анекдотам.

Семантический - это OpenAi Embeddings.  

Анекдоты с сайта http://bashorg.org.
Около 1000.

Поиск - это векторная БД Pinecone.

## Реализация

* Ruby 3.1.2

* Rails 7

* Bootstrap 5
